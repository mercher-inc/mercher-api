## Getting Started

Install the module with: `npm install mercher-api`

```js
var mercher-api = require('mercher-api');
mercher-api.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g mercher-api
$ mercher-api --help
$ mercher-api --version
```


```sh
# creates a browser.js
$ grunt browserify
```
